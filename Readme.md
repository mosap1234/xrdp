# 🚀 سيرفر مجاني بواجهة رسومية XFCE وثيم Windows 11 (جاهز للنشر على Railway)

هذا المستودع يحتوي على إعدادات **Dockerfile** جاهزة لإنشاء سيرفر لينكس مجاني (Ubuntu 22.04) كامل ومزود بواجهة رسومية أنيقة وخفيفة (**XFCE4**)، مخصصة لتبدو تماماً مثل نظام **Windows 11** المظلم. السيرفر يأتي مهيأً ومثبتاً عليه أهم أدوات التطوير والتصفح، وجاهز تماماً للنشر بضغطة زر على منصة **Railway**.

---

## ✨ المميزات الرئيسية للمشروع
* **🖥️ واجهة رسومية كاملة:** استخدام واجهة XFCE4 السريعة والخفيفة على الموارد.
* **🎨 ثيم Windows 11 الاحترافي:** مدمج برمجياً بالثيم والأيقونات والخلفية الرسمية المظلمة لويندوز 11.
* **🌐 متصفح Google Chrome:** مثبت ومعدل ليعمل مباشرة وبأمان بصلاحية الـ Root بدون مشاكل (مع تفعيل خيار `--no-sandbox`).
* **💻 محرر الأكواد VS Code:** جاهز للمطورين لكتابة وتعديل البرمجيات مباشرة من السيرفر.
* **⚡ وصول VNC سريع:** مجهز بسيرفر TigerVNC للوصول لسطح المكتب بدقة عالية Full HD (1920x1080) من أي جهاز (كمبيوتر أو جوال).
* **☁️ متوافق مع Railway:** مهيأ تماماً ليعمل كـ Container مستقر على منصات الاستضافة السحابية.

---

## 🔐 بيانات الدخول الافتراضية
عند الاتصال بالسيرفر، استخدم البيانات التالية:
* **المستخدم (User):** `root`
* **كلمة المرور (VNC Password):** `sudospace`
* **المنفذ الداخلي (Internal Port):** `5901`

---

## 🛠️ خطوات الشرح والنشر على منصة Railway

لإطلاق هذا السيرفر وتشغيله مجاناً، اتبع الخطوات البسيطة التالية:

### 1️⃣ الخطوة الأولى: رفع الكود إلى GitHub
1. قم بإنشاء مستودع (Repository) جديد على حسابك في GitHub (مثلاً باسم: `Free-VNC-Server`).
2. قم برفع ملف الـ `Dockerfile` الخاص بك إلى هذا المستودع.

### 2️⃣ الخطوة الثانية: النشر على Railway
1. توجه إلى موقع Railway وسجل دخولك باستخدام حساب GitHub الخاص بك.
2. اضغط على خيار **New Project** (مشروع جديد).
3. اختر **Deploy from GitHub repo** ثم حدد المستودع الذي أنشأته ورفعت فيه الملف.
4. ستتبنى منصة Railway قراءة الـ Dockerfile وبناء السيرفر تلقائياً خلال دقائق.

### 3️⃣ الخطوة الثالثة: إعداد الاتصال الخارجي (TCP Proxy)
بما أن بروتوكول VNC يعتمد على اتصال TCP وليس HTTP التقليدي، يجب تفعيل منفذ خارجي في Railway:
1. بعد اكتمال بناء السيرفر، اضغط على الخدمة (Service) داخل لوحة تحكم Railway.
2. اذهب إلى تبويب **Settings** (الإعدادات).
3. انزل لأسفل حتى تصل إلى قسم **Networking** (الشبكات).
4. اضغط على **⚡ Generate Proxy** لتوليد رابط اتصال TCP خارجي.
5. تأكد من ربط المنفذ الداخلي (Internal Port) على الرقم `5901`.
6. سيعطيك Railway رابطاً مخصصاً للاتصال الخارجي يشبه هذا: `services-production.up.railway.app:XXXXX` (حيث XXXXX هو رقم المنفذ الخارجي المتغير).

---

## 📺 كيف تتصل بالسيرفر عبر الـ VNC؟

1. قم بتحميل وتثبيت برنامج **VNC Viewer** على جهازك (الكمبيوتر أو الجوال).
2. افتح البرنامج وأدخل رابط الاتصال الخارجي والمنفذ الذي وفرته لك منصة Railway (مثال: `services-production.up.railway.app:12345`).
3. سيطلب منك البرنامج كلمة المرور، أدخل: `sudospace`.
4. استمتع بسيرفرك الشخصي المجاني بواجهة ويندوز 11 الرائعة!

## ⚠️ ملاحظات هامة للمطورين والمشاهدين
* **تشغيل البرامج كـ Root:** تم تعديل اختصارات تشغيل Google Chrome و VS Code داخل الـ Dockerfile لتعمل بسلاسة بصلاحيات الـ Root بدون إظهار رسائل خطأ الأمان التقليدية.
* **دقة الشاشة:** السيرفر مضبوط افتراضياً على دقة `1920x1080` (Full HD)، إذا أردت تعديلها يمكنك تغيير القيمة في السطر الأخير من ملف الـ Dockerfile قبل النشر.
